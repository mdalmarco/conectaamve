# Conecta AMVE

Landing page "Em construção" do Programa Conecta AMVE — portal oficial do programa que conecta pessoas, ideias, inovação e oportunidades em todo o Vale Europeu.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- React + TypeScript
- Tailwind CSS v4
- shadcn/ui
- Framer Motion

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```
